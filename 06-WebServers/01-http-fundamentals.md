- Learn message structure first in [01a-http-basics.md](./01a-http-basics.md).
- Use [01b-http-headers-and-caching.md](./01b-http-headers-and-caching.md) when debugging browser behavior, cache misses, cookies, or cross-origin requests.
- Use [01c-http2-http3.md](./01c-http2-http3.md) for protocol-version comparisons and request lifecycle context.
- Use [01d-http-security.md](./01d-http-security.md) for CORS and transport-aware delivery topics.
- Use [01e-http-troubleshooting.md](./01e-http-troubleshooting.md) during incidents.
- Keep the status-code references and review file bookmarked for quick lookup.
